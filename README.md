# deep-learning-python

## Requirements
- python 3.6
- pipenv 9.0.0

## First
- pipenvのインストール

```
pip install pipenv
```

## Setup
- 仮想環境にpythonのパッケージをインストール

```
pipenv install
```

## Usage
- 実行

```
pipenv run python (target).py
```
